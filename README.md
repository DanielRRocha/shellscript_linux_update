# shellscript_linux_update

This shellscript update linux through the following commands:


- $ sudo apt update -y
- $ sudo apt upgrade -y
- $ sudo apt autoremove
- $ sudo apt autoclean
- $ sudo apt clean




## How to Use

- Open terminal
- Go to the folder with this shellscript
- Use the following command

```bash
  $ ./shellscript_linux_update
```
